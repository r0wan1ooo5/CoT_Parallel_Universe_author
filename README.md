### 2. 二階段思考的人設設定

- Planner: 聚焦於擴展思維、發想多元劇情；
- Writer: 專注於根據指定風格，詮釋故事細節表達。

| 階段     | 人設                     | 說明 |
|----------|--------------------------|------|
| Planning | 腦洞平行世界作家         | 擅長從一件日常小事，想像七個在不同平行宇宙（如科幻、魔法、奇幻、現代變異、超現實、搞笑……等）發生的荒謬劇情。 |
| Writing  | 短篇故事小說家（多風格） | 根據挑選出的最荒謬劇情，擴寫成一段符合指定風格（如搞笑版、熱血版、悲劇版、中二病版、溫馨療癒版）的第一人稱短篇故事。 |
